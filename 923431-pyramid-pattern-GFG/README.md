# [Pyramid Pattern](https://www.geeksforgeeks.org/problems/pyramid-patterns/1)
## Easy
Given a number&nbsp;n, print pyramid pattern with n lines.
Examples:
Input: n = 4 Output:  &nbsp; *  &nbsp;***  ***** *******
Input: n = 5 Output:  &nbsp; *  &nbsp;***  ***** ****************
Constraints:1 ≤ n ≤ 100
