# [Print this pattern!](https://www.geeksforgeeks.org/problems/print-this-pattern0709/1)
## Easy
Given an integer N, you need to print the following pattern.

Example 1:

Input:
N = 3
Output:
33333
32223
32123
32223
33333

Explanation:
When N = 3 then there will be three boundaries.
Outer boundary contains only 3.
Middle boundary contains only 2.
And the inner boundary contains only 1.

Example 2:

Input:
N = 4
Output:
4444444
4333334
4322234
4321234
4322234
4333334
4444444

Your Task:&nbsp;&nbsp;
You don't need to read input. Your task is to complete the function&nbsp;printPattern()&nbsp;which takes an integer N as an input parameter and print the pattern for it.

Expected Time Complexity:&nbsp;O(N2)
Expected Auxiliary Space:&nbsp;O(1)

Constraints:
1 &lt;= N &lt;= 9
