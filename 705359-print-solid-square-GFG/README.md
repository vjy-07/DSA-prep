# [Print Solid Square](https://www.geeksforgeeks.org/problems/print-square-wall-1605682270/1)
## Easy
Given an integer n, print a solid square pattern of size n × n using "* " (a star followed by exactly one space).
Examples: 
Input: n = 5
Output:
* * * * ** * * * ** * * * ** * * * ** * * * *
Explanation: A solid square of size 5 × 5, with each row containing 5 stars followed by a single space.
Input:  n = 4Output:* * * * * * * * * * * * * * * * Explanation: A solid square of size 4 × 4, with each row containing 4 stars followed by a single space.
Constraints:1 ≤ n ≤ 10
