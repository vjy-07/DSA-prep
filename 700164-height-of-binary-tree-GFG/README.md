# [Height of Binary Tree](https://www.geeksforgeeks.org/problems/height-of-binary-tree/1)
## Easy
Given the root of a binary tree, your task is to find the&nbsp;maximum depth&nbsp;of the tree.
Note:&nbsp;The maximum depth or height of the tree is the number of edges in the tree from the root to the deepest node.
Examples:
Input: root = [12, 8, 18, 5, 11]Output: 2Explanation: One of the longest path from the root(node 12) goes through node 8 to node 5, which has 2 edges.
Input: root = [1, 2, 3, 4, N, 10, 5, N, N, N, N, 6, 7]Output: 3Explanation: The longest path from the root(node 1) to a leaf node 6 with 3 edges.
Constraints:1&nbsp;≤&nbsp;number of nodes&nbsp;≤ 3*1040&nbsp;≤ node-&gt;data&nbsp;≤ 105
