# [Inverted Right AngleTriangle](https://www.geeksforgeeks.org/problems/inverted-right-angletriangle-1605691171--104349/1)
## Easy
Given an integer n.&nbsp;Write a program to print the inverted "Right angle triangle" wall.&nbsp;The length&nbsp;of the perpendicular and base&nbsp;is n.
Note:&nbsp;Use string multiplication for python.
Examples:  
Input: n = 4
Output:
* * * * * * * * * *
Explanation: Length of perpendicular and base of triangle is 4 .
Input: n = 3Output:* * * * * *
Explanation: Length of perpendicular and base of triangle is 3 .
Constraints:1 &lt;= n &lt;= 10
