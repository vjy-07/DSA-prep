# [Most Frequent in an Array](https://www.geeksforgeeks.org/problems/most-frequent-element-in-an-array/1)
## Easy
You are given an integer array arr[]. You need to return the element which occurs maximum times in arr[].Note: If multiple such elements exists return the maximum element.
Example:&nbsp;
Input: arr[] = [1, 2, 2, 2, 4, 1]Output: 2Explanation: 2 is most frequent element of this array with 3 occurrences.
Input: arr[] = [1, -5, 8, 1]Output: 1Explanation: 1 is most frequent element of this array with 2 occurrences.
Input: arr[] = [3, 0, 0, 3, 8]Output: 3Explanation: 0 and 3 are two most frequent elements of this array. 3 is the maximum one.
Constraints:1 ≤ arr.size() ≤ 105-105 ≤ arr[i] ≤ 105
